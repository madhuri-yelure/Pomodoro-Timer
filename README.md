# Pomodoro Timer

A responsive Pomodoro Timer web application built using HTML, CSS, and JavaScript to improve productivity and time management. It helps users stay focused by following the Pomodoro Technique with work and break sessions.

## Features

- Start, pause, and reset the timer
- Countdown timer for focused work sessions
- Short and long break support
- Simple and user-friendly interface
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://madhuri-yelure.github.io/Pomodoro-Timer/

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
